# Student Management API

## Description
This project is a simple REST API built with Node.js and Express that allows users to manage student records.

## Features
- View all students
- View a single student
- Add a student
- Update a student
- Delete a student

## Technologies Used
- Node.js
- Express.js

## How to Run

Install dependencies:

npm install

Start the server:

npm start

Open browser:

http://localhost:3000
